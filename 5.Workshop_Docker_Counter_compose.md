## Ajouter la dépendance à postgresql
- Dans votre fichier requirements.txt (tout en bas, même si peu importe où), ajoutez la référence à :
```
psycopg2-binary
``` 
- Relancez votre commande pour installer les dépendances :
```
python -m pip install -r ./requirements.txt
```

## Modifier le code
- Code du main.py :
```py
import os
import psycopg2
from counter import add_one_to_count, remove_one_to_count

def connect_to_db():
    conn = psycopg2.connect(
        host=os.getenv("DB_HOST", "localhost"),
        database=os.getenv("DB_NAME", "results"),
        user=os.getenv("DB_USER", "postgres"),
        password=os.getenv("DB_PASSWORD", "postgres")
    )
    return conn


def main():
    conn = connect_to_db()
    cursor = conn.cursor()

    count = add_one_to_count(0)
    print('Affichage du compteur après ajout : ' + str(count))
    cursor.execute(
        "INSERT INTO counter_results (actual_value) VALUES (%s)",
        (count,)
    )
    print('Saved in DB')

    count = remove_one_to_count(50)
    print('Affichage du compteur après retrait : ' + str(count))
    cursor.execute(
        "INSERT INTO counter_results (actual_value) VALUES (%s)",
        (count,)
    )
    print('Saved in DB')

    cursor.execute("SELECT actual_value, updated_at FROM counter_results")
    rows = cursor.fetchall()

    for row in rows:
        print(f"Result: {row[0]} | Date: {row[1]}")

    cursor.close()
    conn.close()

main()

```
- Ajouter un fichier d'initialisation de la base de données (nommé init.sql, à la racine aussi) :
```sql
CREATE TABLE IF NOT EXISTS counter_results (
    id SERIAL PRIMARY KEY,
    actual_value INTEGER NOT NULL,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## Ajouter un Docker compose
- Le but du docker compose est d'orchester (Créer, gérer la persistance des données, gérer les communications...) de nos différents conteneurs.
- Voici le contenu du docker-compose.yaml que vous devez créer à la racine du projet.
```yaml
services:
  db:
    image: postgres:16
    container_name: counter-db
    environment:
      POSTGRES_DB: ${DB_NAME}
      POSTGRES_USER: ${DB_USER}
      POSTGRES_PASSWORD: ${DB_PASSWORD}
    ports:
      - "5432:5432"
    volumes:
      - postgres_data_counter:/var/lib/postgresql/data
      - ./init.sql:/docker-entrypoint-initdb.d/init.sql:ro
    healthcheck:
      test: ["CMD-SHELL", "pg_isready -U postgres -d results"]
      interval: 5s
      timeout: 5s
      retries: 10

  app:
    build:
      context: .
      dockerfile: Dockerfile
    container_name: counter-app
    depends_on:
      db:
        condition: service_healthy
    env_file:
      - .env
    command: tail -f /dev/null

volumes:
  postgres_data_counter:
```
- Démarrer vos conteneurs via docker-compose avec la commande : 
``` docker compose up```
- Entrer dans le conteneurs correspondant à counter-app : 
``` docker exec -it counter-app bash ```
- Lancer votre script python : 
``` python app.py ```

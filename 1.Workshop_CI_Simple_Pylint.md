# Workshop - toute petite intégration continue avec python et pylint
- Créer un repo github avec un .gitignore pour python et un fichier readme.md (même vide, peu importe)
- Créer un fichier app.py avec le code minimal de python pour faire une simple calculatrice (add suffit)
```py
def add(first, second):
    return first + second

add(2,5)

```
- Créer un fichier à la racine : "requirement.txt" avec juste pylint dedans :
```
pylint
```
- Exécuter la commande (à la racine) ```pip install -r requirements.txt```
- Ajouter un dossier .github et un sous dossiers workflows.
- Ajouter le fichier yaml minimal pour faire passer pylint (vous avez un modèle de fichier simple dans les workflows proposés par github action)
- Faire tourner votre CI en faisant un nouveau add / commit / push sur main.

## Si vous avez des remontés de pylint
- Ajouter un fichier .pylintrc à la racine avec ce contenu : 
```
[MESSAGES CONTROL]
disable=
    C0114,
    C0116
```
